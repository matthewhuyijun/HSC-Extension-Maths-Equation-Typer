Header Functions Guide

Copy Button

Copies the LaTeX code to your clipboard. You can also click the Rendered Input, Raw LaTeX Input, or LaTeX Preview labels to copy your expressions.

GitHub

Opens the project repository on GitHub where you can view the source code, report issues, or contribute to the project.

Shortcuts

View all available keyboard shortcuts for the MathLive editor. Learn how to quickly input mathematical symbols and expressions using keyboard commands.

Hide Header

Toggles the header visibility. When enabled, the header is hidden to maximize your workspace. A small button will appear to show the header again.

Pure LaTeX Mode

Switches between rendered math view and pure LaTeX code view. When enabled, you'll see the raw LaTeX code instead of the formatted mathematical expression.

Theme Switcher

Change the appearance theme: Light mode (sun icon), Dark mode (moon icon), or System mode (monitor icon) which follows your device's theme preference.
