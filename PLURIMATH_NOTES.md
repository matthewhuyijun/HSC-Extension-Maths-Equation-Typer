
# Plurimath Notes

- Repository: https://github.com/plurimath/plurimath (Ruby gem).
- Purpose: provides conversions between LaTeX, MathML 3/4, AsciiMath, UnicodeMath, and OMML using a shared AST.
- CLI syntax from README: `plurimath convert [options]` after installing the gem (`gem install plurimath`).
- Useful inspiration: UnicodeMath + MathML conversion parity for clipboard features we are adding to the web app.
