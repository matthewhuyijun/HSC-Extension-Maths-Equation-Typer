## 🎯 核心功能

这是一个 **LaTeX 到 Word 公式转换器**：

```
LaTeX 输入              Word 公式输出
─────────────────────  ─────────────────────
\frac{a}{b}        →   (a)/(b)
\sqrt{x}           →   √(x)
\int_{0}^{1} x dx  →   ∫_(0)^(1) 〖x〗 dx
\sum_{i=1}^{n} i   →   ∑_(i=1)^(n) i
```
